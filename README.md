Descripción general del sistema:  Este sistema simula una caja de supermercado, permitiendo registrar productos, agregarlos a un carrito de compras, calcular el costo total de cada venta y generar estadísticas, como el producto más vendido.

Se utilizo en el codigo la inteligencia artificial Chat Gpt en el diseño y logica de la funcion "estadisticas".

Instrucciones de ejecución: 

0 Salir del programa
	
		Cierra el programa

1. Registrar producto
	
		Se debe ingresar un Código entre 001 y 100 ej (032, 099, 027)
	
		Se debe ingresar un nombre por ej (pan, harina, sal)
	
		se debe ingresar un precio en entero ej (1200, 3000, 1460)
	
		se debe ingresar un stock ej (20, 50, 39)

2. Mostrar productos
	
   		Se muestran todos los productos ya precargados por el Código y los ingresados por el usuario

3. Agregar al carrito.
   
   		Se debe ingresar un Código ya cargado por el usuario o uno precargado
	
		Se debe ingresar la cantidad ej (3, 5, 8)

4. Mostrar carrito
	
		Se muestran los productos dentro del carrito junto a sus datos

5. Finalizar Compra
	
		Se debe elegir o no un descuento
		
		Se carga el carrito y se muestra el ticket, para despues vaciar el carrito

  
7.  Estadistica

    	Se muestran los productos registrados, unidades vendidas, dinero recaudado, productos mas vendidos y top de productos
